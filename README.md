# SmartCalc – OmniSuite
### CLI Smart Calculator & Shopkeeper Billing System

> **SmartCalc – OmniSuite** is a modern browser-based productivity application that combines a scientific calculator, a shopkeeper billing system, and a simulated command-line interface (CLI) into a single lightweight dashboard.

Built entirely using **HTML5, CSS3, and Vanilla JavaScript (ES6 Modules)**, the project demonstrates modular software architecture, responsive UI design, state management, local storage, and object-oriented programming concepts—without relying on any external libraries or frameworks.

---

# Project Overview

SmartCalc – OmniSuite was designed as an educational yet production-inspired application to showcase modern frontend development principles using only core web technologies.

The application focuses on:

- Clean SaaS-inspired dashboard UI
- Modular JavaScript architecture
- Responsive layouts
- Persistent data storage
- Interactive command-line simulation
- Shop billing automation
- Invoice generation
- Lightweight browser performance

# Key Features

## Smart Calculator

A fully functional calculator supporting everyday mathematical operations with memory registers.

### Features

- Basic Arithmetic Operations
- Decimal Calculations
- Percentage
- Memory Functions
  - MC
  - MR
  - M+
  - M-
- Live Display
- Calculation History
- Keyboard Support
- Local Storage Persistence

---

## Shopkeeper Billing System

A lightweight Point-of-Sale (POS) interface for generating customer bills.

### Features

- Add Multiple Products
- Quantity Management
- Price Calculation
- GST Support
  - 5%
  - 12%
  - 18%
  - 28%
- Discount Coupons
- Profit Margin Calculation
- Auto Grand Total
- Remove/Edit Items
- Search & Filter Products
- Sort Items
- Persistent Billing Data

---

## Interactive CLI Terminal

A simulated command shell allowing users to manage billing operations using text commands.

Supported commands include:

```bash
help
add
delete
update
clear
history
total
print
exit
```

The terminal provides a realistic developer experience by interacting directly with the billing engine.

---

## Invoice Generator

Generate clean printable invoices directly from the browser. Features include :
- Customer Details
- Product Table
- GST Breakdown
- Discount Summary
- Grand Total
- Printable Layout
- HTML-based Invoice
- Browser Print Support
